博客的搭建

环境的准备

1.安装node.js

  • 下载：官方地址：https://nodejs.org/zh-cn/
  • 直接安装：无脑next下一步
  • 校验是否安装成功：node -v

2.安装镜像源cnpm

  • 利用npm安装：npm install -g cnpm --registry=https://registry.npm.taobao.org

3.安装hexo博客框架

  • 利用cnpm安装：cnpm install -g hexo-cli
  • 校验：hexo -v

4.安装git

  • 下载：官方地址:https://git-scm.com/
  • 直接安装：无脑next下一步
  • 校验是否安装成功：鼠标右会多两个git选项

5.创建博客放置文件夹

  • 选择一个你想存放博客的地方创建一个文件夹

安装博客

  1. 在你文件夹下右键点击Git Bash Here进入git命令行

  2. 输入**hexo init **命令

  3. 出现Start blogging with Hexo!说明安装成功

博客使用

基本使用

  1. 先使用hexo new “xxxx”创建一篇文章（创建一篇博文（生成的md文件在source_posts路径下））
  2. 然后再hexo clean清理缓存
  3. hexo generate/hexo g 生成静态文件
  4. hexo s启动博客
  5. 在浏览器中输入http://localhost:4000/即可访问
  6. 部署网站：hexo d后续会使用

更多具体命令参考https://hexo.io/zh-cn/docs/commands.html
