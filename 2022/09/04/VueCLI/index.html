VueCLI

介绍

CLI是Command-Line Interface，翻译为命令行界面，但是俗称脚手架。
Vue CLI是一个官方发布vue.js项目脚手架，使用vue-cli可以快速搭建Vue开发环境以及对应的webpacki配置，

安装

  1. 全局安装@vue/cli：npm install -g @vue/cli
  2. 切换到要创建项目的目录进行创建：vue create XXXX (vue cli3)/ vue init webpack 项目名称（vue cli2）
  3. 启动项目：npm run serve

相关配置

方式一：通过vue ui修改

  1. 进入vue ui后导入项目

  2. 可以管理相应的依赖和插件

  3. 在配置项，配置相关配置

方式二：直接创建配置文件配置

  1. 创建vue.config.js文件（名称必须固定）

  2. 直接在文件中修改配置文件
